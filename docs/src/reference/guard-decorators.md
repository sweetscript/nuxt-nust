# Guard decorators

## `@UseGuards` {#@UseGuards}

Decorator that add guard to method route handler

#### Signature

```typescript
@UseGuards(guards:...NustGuard[])
```

#### Parameters

| Parameter   | Type          | Required | Default | Description                    |
|-------------|---------------|----------|---------|--------------------------------|
| `guards`    | `NustGuard[]` | Yes      | -       | The guard classes or instances |
